Highlights

Library Terminal

A program utilizing an interface that allows the librarian

the ability to create, delete, or checkout items from the library.

This applications main goal is to make the duties of the librarian

simpiler, and easier when dealing with large amounts of inventory.

Rock, Paper, Scissors

An application allowing the user to play a game of rock, paper, scissors

with the computer. The user can input their name, and select one of two

opponents. Each opponent has their own style of play.
